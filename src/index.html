MY PROTFOLIO

AI Strategy Competition Platform

The AI Strategy Competition Platform is an online competition platform for board games. Students can construct their AI strategy using C or Java to compete with each other on the platform. A ranking system is also provided by the platform to motivate the students to improve their AI strategies. With motivation to achieve higher rankings, students can practice programming skills while improving their artificial intelligence strategies.

API Research Infrastructure

The API Research Infrastructure is an infrastructure software that provides an interface for API researchers to observe and analyze API usage data. The infrastructure software is launched in 2020 and accumulates over a million of API usage data.

Play and Learn Together

Play and Learn Together is a web application that 'gamifies' the process of programming learning. It is launched in 2020 and serve as the learning website for the Introduction to Computers Course at National Cheng Kung University. It is one of my winning projects in DBSE Hack Day.
