API Research Infrastructure

Infrastructure software for API Researchers

About the Project

This project aims to expand the research area in API in the Database and Software Engineering Lab (DBSE). The API Research Infrastructure can automatically collect API usage data and provide an interface for API researchers to observe and analyze API usage data. The infrastructure software is launched in 2020 and accumulates over a million of API usage data. Several API researchers in the DBSE Lab benefited from this infrastructure and complete their API-related research including my thesis and a paper (Link) published on Information and Software Technology (IST) by another lab member.
