Play and Learn Together

Web Application for Programming Education

About the Project

This project aims to provide a 'gamified' way for programmers to learn and improve their programming skills. The Play and Learn Together web application turns different programming problems into challenges. By taking the challenges, Users can learn programming knowledge and improve their programming skills. Users can also share their ideas on the forum and work together to overcome the challenges. Since the project faces unclear constraints and requirements, this project applies scrum methodology to iterate quickly and communicate changes with stakeholders. The Play and Learn Together web application is launched in 2020 and serve as the learning website of Introduction to Computers Course in National Cheng Kung University in 2021. The success of the website earned all project members an Excellent Teaching Assisting Award in 2022.
